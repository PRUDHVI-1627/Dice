# 🎲 Dice Roll Web App

A simple and interactive dice roll web application built using HTML, CSS, and JavaScript.  
Click the button to generate a random number between 1 and 6 — just like rolling a real dice!

---

## 🚀 Features

- 🎯 Generates random numbers from 1 to 6
- 🖱️ Button click interaction
- ⚡ Instant result display
- 💡 Beginner-friendly project

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📂 Project Structure

Dice/
│── index.html
│── index.js
│── style.css

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/PRUDHVI-1627/Dice.git

2. Open the folder

3. Run the project:
   Open index.html in your browser

---

## 🎮 How to Use

- Click the "Roll" button  
- A random number between 1–6 will be displayed

# 🎲 Dice Roll Web App

A simple and interactive dice roll web application built using HTML, CSS, and JavaScript.  
Click the button (or press **Space**) to roll the dice and get a random number from 1 to 6!

---

## 🚀 Features

- 🎯 Generates random numbers from 1 to 6
- 🖼️ Dice face images that change on every roll
- 🖱️ Button click interaction
- ⌨️ Keyboard support — press **Space** to roll
- 🎞️ Smooth roll animation
- 📜 Roll history — tracks your last 5 rolls
- 💡 Beginner-friendly project

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript


2. Open the folder and run the project:
   ```
   Open index.html in your browser
   ```

---

## 🎮 How to Use

- Click the **Roll Dice** button, or press **Space**
- The dice image will animate and land on a random face
- The result is shown below the dice
- Your last 5 rolls are saved in the history section
